# Screenshots

`overview.svg` is a lightweight **mockup** of the UI so the README renders visuals
without a running browser.

To use real screenshots instead:

1. Run the app (`docker compose up -d`) and open http://localhost:8080.
2. Capture the start screen and a question card (⌘⇧4 on macOS).
3. Save them here as `overview.png` and `answering.png`.
4. Update the image links in `README.md` / `README.zh-TW.md` to point to the PNGs.
